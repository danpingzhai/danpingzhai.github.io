Linux 目录（directory）和常见目录操作命令

一些概念
  • 目录 (directory)：通俗的讲，一个目录对应着Windows中文件夹的概念，但其实更加广泛。
  • 操作系统中的目录/文件夹/directory/ 是一级一级的，每一级目录直接用/隔开。比如有个目录叫 'my-folder'，它的下面有一个子目录叫 'sub-folder'， 那么子目录的路径就是 my-folder/sub-folder。
  • 路径 path: 表示每一个文件/文件夹的“位置或者地址”。
一些常见的 directory 和命令
  • 非常重要 ./ 表示“当前工作目录”(working directory)。 你可以想象你正打开一个windows文件夹，那么此时的 ./ 就是你正在打开的这个文件夹。 在命令行(command line)中，当前目录./ 就是你目前所处的目录位置。可以用 pwd (print working directory) 来打印出当前目录。
  • ~/ 表示“用户目录”，它不是一个实际存在的目录，而是一种快捷访问的方式。 绝大部分（或者几乎所有）操作系统都是多用户的（虽然初级用户可能只有一个用户在使用）。每一个用户都有自己的“用户目录”。 以 Windows 操作系统为例，用户目录在 `C:/Users` 下，假设操作系统中有 zhangsan，lisi 等多个用户，那么他们的用户目录分别是 `C:/Users/zhangsan`, `C:/Users/lisi`。 假设现在是 zhangsan 登录进了这个操作系统，那么他的用户目录 `~/` 就是 `C:/Users/zhangsan`。 所以说 `~/` 这个目录并不是实际存在的一个目录，而是快捷访问 `/home/zhangsan` 的一种方式。 在很多操作系统中用户目录下都有 "Desktop（桌面）"，“Document（文档）”，“Pictures（图像）”等子目录，这些子目录的路径 就是~/Desktop, ~/Document, ~/Pictures等等。
  • cd 命令用来切换 working directory (非常重要 )。 cd 命令后面接一个参数，表示你想切换到的目标路径。 比如 cd ~/ 切换到的“用户目录”。
  • 比如说在很多操作系统中用户目录下都有 "Desktop（桌面）"，“Document（文档）”，“Pictures（图像）”等子目录，这些子目录的路径 就是~/Desktop, ~/Document, ~/Pictures等等。

Linux commands

一般一个完整的命令 (command) 由“命令名”, “options” 和“参数”组成, 一般 options 都以“-”开头. 有的命令不需要 options, 有的命令既不需要 option, 也不需要 argument.

command -options arguments

例如, 删除一个名字叫 "photos" 的目录所需的命令为rm -r photos, 其中

  1. rm为命令名, rm命令用于删除文件或目录;
  2. -r 为 option, 表示“recursive”, 如果要删除目录, 需用此option;
  3. photos 为“argument”, 指被删除的目录名字.
又例如, 删除一个名字叫 "photos.jpg" 的文件所需的命令为rm photos.jpg, 这个命令不需要 options. 用于打印当前的工作路径的命令为pwd, 这个命令既没有option, 有没有 argument.

常用的命令

  • cd: change directory
  • cp: `cp source-file.xx target-file.xx`
  • rm: romove
  • ls: list all files under current directory
  • pwd: print working directory
  • mkdir:"make directory"

Tags

典型的 html tags 有 p, h1, h2, h3, span, section, img. 一个 tag 由 开头, 以 结束, 第一个叫 opening tag, 第二个为 closing tag. 绝大部分 tags 都需要 closing tag, 有少数 tags 只需要 opening tag (例如 img).

tag display

  • block-level:h、p、div、ul、ol、li,独占一行,height、weight和margin、padding均可控制;文字类tag中不能放block-level(p中不能放div);
  • inline-level:a、span、strong、b、em、i、del、s、ins、u;一行可以显示多个；直接设置高宽无效,默认宽度是内容宽度;行内元素只能放文本或其他行内元素;
  • inline-block:img、input、td和相邻元素在一行上,中间会有空白缝隙,一行可以显示多个;default weight是内容本身的weight;height、margin、padding都可以控制

change to different display: display:inline 、display:block、display:inline-block

Cascaded Style Sheet

There are three types of CSS:

  1. Inline: styles defined by 'style' attribute of an element inside the open tag of the element. Example: ❬p style='font-size: 2rem;'❭ xxxx ❬/p❭
  2. internal: styles defined inside ❬style❭ ❬/style❭in the head of a HTML document.
  3. external: defined in a seperate .css file and is linked into a HTML document.

DOM:doucument object model

DOM 是Javascript操作网页的接口, 可以将网页转化为Javascipt对象,从而用脚本进行各种操作。 DOM只是一个接口规范,可以用各种语言实现。 严格说DOM 不是Javascript语法的一部分, 但DOM是Javascrpt最常见的任务,离开了DOM,javascrpt就无法控制网页。 DOM最小单位是节点(node),有7种类型:

  1. document:文档树的顶层节点
  2. document type: doctype标签
  3. element:网页的各种html标签
  4. attribute:网页元素的属性
  5. text: 标签之间或标签包含的文本
  6. comment:注释
  7. documentfragment:文档的片段
